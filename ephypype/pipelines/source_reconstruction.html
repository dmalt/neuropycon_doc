Source reconstruction pipelineÂ¶

The inverse solution pipeline performs source reconstruction starting either from raw/epoched data (.fif format) specified by the user or from the output of the Preprocessing pipeline (cleaned raw data).

The output is the source reconstruction matrix that can be used as input of the Connectivity pipeline.

The pipeline is defined by the function create_pipeline_source_reconstruction

The three Nodes of the inverse solution pipeline (LF_computation, create_noise_cov, inv_solution) wrap the MNE Python functions that perform the source reconstruction steps.

In particular:

  • Lead Field Node: compute the Lead Field matrix
  • Noise Covariance Node: compute the noise covariance matrix
  • Inverse Solution Node: estimate the time series of the neural sources on a set of dipoles grid

Before to use this pipeline, we need for each subject a template MRI or the individual MRI data.
