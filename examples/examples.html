Examples¶

All example scripts are based on the sample datasets provided by OMEGA project. In particular, we used the datasets organized according to the BIDS specifications (Brain Imaging Data Structure): after the registration to the OMEGA website, the sample datatasets to use with the scripts can be downloaded from the OMEGA link.

All example scripts need a params file where to set

  • the list of subjects
  • the data path
  • the parameters related to the preprocessing pipeline
  • the parameters related to the connectivity pipeline
  • the parameters related to the source reconstruction pipeline
